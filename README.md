# Wrapper Orogrande per GitHub Pages / ViPlex
Pubblica questa cartella su **GitHub Pages** per ottenere un URL da usare in ViPlex come "Web Page".
Visualizza `https://www.orogrande.it/landing-screen` ridimensionato alla risoluzione desiderata (default 336x252).

## Parametri
- `?w=336&h=252` → dimensione LED
- `&url=https://www.orogrande.it/landing-screen` → URL sorgente (default)
- `&mode=cover|contain|stretch` → modalità adattamento (default: cover)
- `&base=1920x1080` → viewport virtuale

## Esempio URL
`https://tuonome.github.io/orogrande-led/?w=336&h=252&url=https://www.orogrande.it/landing-screen&mode=cover`

Inserisci questo link nella tua Solution ViPlex (Web Page) con risoluzione 336×252.
